Imputing missing values

naNumb = sum(is.na(data[1]))

There data contains 2304 missing values. Analyzing the file it stands out that thesw are not single missing values, but there are always complete days that are missing.

A strategy to fill out the missing values in this case could be to fille them with the mean of the values measured on the same weekday and on the same time interval.

Are there differences in activity patterns between weekdays and weekends?

# factor for weekday vs weekend

dayarray = weekdays(data2$date)

weekday <- function(x) {
    if (x=="Sunday"|x=="Saturday") {
        return("weekend")
    } else {
        return("weekday")
    }
}
